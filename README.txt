Protótipo V8.1 - correção de atualização do PWA

Esta versão preserva a paleta e o ícone da V8 e corrige o cache do aplicativo.
Agora, quando houver internet, o app procura primeiro a versão mais recente publicada no GitHub Pages. Assim, mudanças futuras de cores, ícones e telas passam a aparecer sem ficarem presas na versão antiga.

Para publicar:
- Substitua app.js e sw.js no repositório.
- Se quiser garantir que o ícone também seja o novo, envie novamente a pasta icons.
- Faça o commit: "Corrige atualização automática do app".
